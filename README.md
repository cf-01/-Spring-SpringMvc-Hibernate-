# 基于spring+spring mvc+hibernate的仓库管理系统 


#### 介绍
本项目是一个 spring + spring mvc + hibernate整合的小项目，源自本人代做的作业，适合用来提交作业。
提供了完整的增删改查案例，前端使用LayUI+BootStrap+Jsp完成
使用了部分ajax请求

#### 软件架构
软件架构说明
spring + spring mvc + hibernate + jsp + LayUI


#### 安装教程

1.  创建数据库，导入数据库文件
2.  IDEA添加Tomcat运行
3.  访问localhost:8080进入登录页面
